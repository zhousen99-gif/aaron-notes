📰 事件背景 📰 Incident Background

2026年3月31日，Anthropic 在发布 Claude Code 的 npm 包时，因构建配置失误将 source map 文件（.map） 包含在发布包中。.map 文件的 sourcesContent 字段直接嵌入了完整原始 TypeScript 源码，导致约 1,906 个文件、800K+ 行代码全部暴露。 On March 31, 2026, Anthropic accidentally included source map files (.map) in the published npm package of Claude Code. The sourcesContent field in the .map files contained the complete original TypeScript source — roughly 1,906 files and 800K+ lines of code were exposed.

泄露原因：Bun 打包工具默认生成 source map，团队未在 .npmignore 中排除 *.map 文件，也未配置 bundler 关闭生产环境 source map 生成。 Root cause: Bun bundler generates source maps by default. The team failed to exclude *.map in .npmignore or disable source map generation for production builds.

讽刺的是，Claude Code 内部有一套名为 Undercover Mode 的子系统，专门防止 AI 在 commit / PR 中泄露内部信息——然而源码本身却通过一个 .map 文件暴露了。 Ironically, Claude Code has an internal subsystem called Undercover Mode specifically designed to prevent the AI from leaking internal info in commits / PRs — yet the source itself was exposed via a .map file.

🏗️ 整体架构 🏗️ Overall Architecture

Claude Code 不是一个简单的 CLI 工具，而是一个完整的终端应用程序。它用 React（通过自定义 Ink 终端渲染器 + Yoga flexbox 布局引擎）驱动 UI，核心由查询引擎、工具系统、权限系统和多 Agent 编排构成。 Claude Code is not a simple CLI tool — it's a full terminal application. It uses React (via a custom Ink terminal renderer + Yoga flexbox layout engine) to drive the UI, with a query engine, tool system, permission system, and multi-agent orchestration at its core.
