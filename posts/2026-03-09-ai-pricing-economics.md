今天在对比各家大模型 API 定价的时候，看到一些数字：Claude Opus 4 的 output 定价是 $75/M tokens，而通义千问 Turbo 是 ¥0.6/M tokens。换算过来，差了大约 **500 倍**。

500 倍是什么概念？大概相当于一杯手冲精品咖啡和一杯白开水的价差。但问题是，对于日常 80% 的使用场景，这两杯水解渴的效果差距远没有 500 倍。

这让我开始思考一个更底层的问题：**AI 算力的定价，到底在定什么的价？**

## 不是成本定价，是身份定价

传统软件的定价逻辑很简单——边际成本趋零，所以定价基本反映的是品牌溢价和用户支付意愿。但大模型 API 不一样，它的边际成本不为零，每一次推理都在烧 GPU。按理说定价应该更接近成本。

然而现实是，同一个问题，GPT-5.4 收你 $2.5/M input + $15/M output，DeepSeek V3 只收 ¥2/M input + ¥8/M output，差了将近 10 倍。它们跑的是不同复杂度的模型，但都是 transformer 架构，都在 GPU 上推理。成本端的差距可能是 2-3 倍，定价端的差距却是 10-50 倍。

这说明当前大模型的定价更像是一种**身份定价**——你用 Opus 不只是用一个模型，你在为"最聪明的 AI"这个标签付费。就像奢侈品，功能价值和情感价值是分开的。

## 中国式价格战：技术降本 + 战略亏损

中国市场的故事完全不同。2024 年字节豆包率先把定价打入"厘时代"（0.0008 元/千 tokens），然后百度、阿里、DeepSeek 纷纷跟进。Benedict Evans 在 2026 年 2 月的分析中指出，Meta 和 Amazon 等巨头的策略就是**把模型层商品化（commoditize）**，以接近边际成本的价格销售，从而把竞争拉到应用层。中国的大模型厂商提前实践了这个逻辑。

不过有意思的是，2025 年下半年开始，价格战出现了**回调**。DeepSeek V3 结束优惠期后价格翻了几倍，智谱、月之暗面等也停止了降价。原因很直接：Agentic AI（智能体）的兴起让单次交互的 token 消耗暴涨，厂商扛不住了。价格战可以抢市场，但不能永远亏下去。

## 真正的趋势：每年降一个数量级

抛开短期的价格战和回调，长期趋势是确定的。Epoch AI 的研究显示，自 2024 年 1 月以来，**同等能力下的推理成本每年下降 50 倍**（中位数），最快的方向甚至达到年降 900 倍。OpenAI 自 GPT-3 发布以来，API 定价已经累计下降了 97%。

这个速度比摩尔定律（每两年翻一倍）快得多。驱动力来自三个方向：模型架构优化（如 DeepSeek 的 Sparse Attention 直接砍半计算量）、推理基础设施进步（智能缓存带来 5-10 倍 GPU 成本节约）、以及市场竞争带来的主动降价。

## 所以，定价在收敛

我从今天的对比中得出一个判断：**大模型定价正在从"奢侈品逻辑"向"自来水逻辑"转变。**

现阶段的 500 倍价差是不可持续的。随着开源模型（DeepSeek、Qwen、Llama）不断逼近闭源模型的能力上限，Opus 的定价权会被持续侵蚀。Benedict Evans 说得很清楚：「商品化的基础设施长期不会有超额利润；如果模型也变成商品，同样如此。」

这对用户意味着什么？**现在选模型，应该按任务复杂度分级，而不是无脑用最贵的。** 简单任务用 Haiku 或 Qwen-Turbo（几乎免费），中等任务用 Sonnet 或 GPT-5.4（每天几毛钱），只有真正需要深度推理的场景才值得上 Opus。

500 倍的价差不是因为 Opus 贵，而是因为 Turbo 类模型已经便宜到接近免费。智能正在自来水化——打开水龙头就有，你只需要为高品质的矿泉水额外付费。
